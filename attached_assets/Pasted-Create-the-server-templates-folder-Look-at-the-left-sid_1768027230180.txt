Create the server/templates folder

Look at the left sidebar (File Explorer)

Find the folder named server

If you don’t see it, expand the root /

Right-click on server

Click “New Folder”

Name the folder exactly:

templates


(all lowercase, no spaces)

You should now see:

server/
  templates/

3️⃣ Create FormQAR-054_C.json

Right-click on the new templates folder

Click “New File”

Name the file exactly:

FormQAR-054_C.json
